# Assets

Ce dossier contient les assets de l'application mobile :

- `icon.png` : Icône de l'application (1024x1024px)
- `splash.png` : Image de splash screen
- `adaptive-icon.png` : Icône adaptative pour Android
- `favicon.png` : Favicon pour le web

Pour générer ces assets, vous pouvez utiliser :
- [Expo Asset Generator](https://www.npmjs.com/package/@expo/asset-generator)
- Ou créer manuellement les images aux dimensions requises
